## 一、项目概述

### 目标
基于《周易》金钱卦（六爻占卜）体系的在线自动占卜工具。用户提问→摇卦→得本卦→AI 解读，完整覆盖占卜流程。

### 参考体系
- **蓍草法原理**：[Humoonruc/auto-divination](https://github.com/Humoonruc/auto-divination)
- **六爻数据**：《周易》通行本（王弼注本、朱熹《周易本义》）
- **卦序**：序卦传体系，六十四卦依《易传·序卦》排列

### 技术栈
- 纯前端（HTML + CSS + Vanilla JS），无框架依赖
- GitHub Pages 托管，GitHub Actions 自动部署
- AI 追问功能预留接口（待接入）

## 四、摇卦算法（核心规范）

### 蓍草法 — 铜钱法实现

```javascript
// 正面(花/heads)=1, 背面(字/tails)=0
// code = coin1 + coin2 + coin3 + 6

if (p === 3) { 老阳○, 变爻 }
else if (p === 2) { 少阴--, 不变 }
else if (p === 1) { 少阳—, 不变 }
else { 老阴×, 变爻 }  // p === 0
```

| 正面数 p | code值 | 属性   | 符号 | 变动 |
|---------|--------|--------|------|------|
| 3个 | 9 | 老阳 | ○ | 变爻 |
| 2个 | 8 | 少阴 | -- | 不变 |
| 1个 | 7 | 少阳 | — | 不变 |
| 0个 | 6 | 老阴 | × | 变爻 |

### 卦象二进制编码规则

**八卦符号对应二进制：**

| 符号 | 名称 | 二进制 | 五行 |
|------|------|--------|------|
| ☰ | 乾 | 111 | 金 |
| ☱ | 兑 | 110 | 金 |
| ☲ | 离 | 101 | 火 |
| ☳ | 震 | 100 | 木 |
| ☴ | 巽 | 011 | 木 |
| ☵ | 坎 | 010 | 水 |
| ☶ | 艮 | 001 | 土 |
| ☷ | 坤 | 000 | 土 |

**爻位 → 二进制位：**

```
Y[0] = 初爻 (bit 0, lower trigram bit 0)
Y[1] = 二爻 (bit 1, lower trigram bit 1)
Y[2] = 三爻 (bit 2, lower trigram bit 2)
Y[3] = 四爻 (bit 3, upper trigram bit 0)
Y[4] = 五爻 (bit 4, upper trigram bit 1)
Y[5] = 上爻 (bit 5, upper trigram bit 2)
```

**gC() 函数实现：**

```javascript
function gC() {
    let l = '', u = '';
    for (let i = 0; i < 3; i++) l += Y[i].sym === '—' ? '1' : '0';  // lower trigram
    for (let i = 3; i < 6; i++) u += Y[i].sym === '—' ? '1' : '0';  // upper trigram
    return u + l;  // upper bits 在前, lower bits 在后
}
```

**hexagrams.js 键规则：** upper(bit5-0) + lower(bit2-0)

### 六爻顺序规范

- 摇卦时：初爻先记录，依次到上爻（由下而上）
- 卦象展示：上爻在上、初爻在下（标准卦象格式）
- `.yaoLines`：flex-direction: column，迭代 5→0
- `.lines`（爻列表）：flex-direction: column-reverse

### 之卦（变卦）计算

```javascript
// 本卦变爻翻转 → 之卦
function getFutureCode(code) {
    // 老阳(9) ↔ 老阴(6)，少阳(7) ↔ 少阴(8)
    if (code === 6 || code === 9) return 15 - code;
    return code;
}
```
